## kp clusterstack delete

Delete a cluster stack

### Synopsis

Delete a specific cluster-scoped stack from the cluster.

```
kp clusterstack delete <name> [flags]
```

### Examples

```
kp clusterstack delete my-stack
```

### Options

```
  -h, --help   help for delete
```
