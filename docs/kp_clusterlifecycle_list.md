## kp clusterlifecycle list

List cluster lifecycles

### Synopsis

Prints a table of the most important information about cluster-scoped lifecycles in the cluster.

```
kp clusterlifecycle list [flags]
```

### Examples

```
kp clusterlifecycle list
```

### Options

```
  -h, --help   help for list
```
