## kp clusterstack list

List cluster stacks

### Synopsis

Prints a table of the most important information about cluster-scoped stacks in the cluster.

```
kp clusterstack list [flags]
```

### Examples

```
kp clusterstack list
```

### Options

```
  -h, --help   help for list
```
